Подключаемые библиотеки

- `pandas` — работа с табличными данными
- `numpy` — численные операции
- `matplotlib` — построение графиков
- `seaborn` — статистические визуализации

Методы предобработки данных

1. Проверка типов данных — `df.dtypes`, `df.info()`
2. Обнаружение пропусков — `df.isnull().sum()`
3. Преобразование типов — `astype('category')` для категориальных переменных
4. Статистическое описание — `df.describe()`
5. Визуализация — гистограммы, scatter plot, boxplot, heatmap
